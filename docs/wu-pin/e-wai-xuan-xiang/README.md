# 额外选项

以默认配置为例:&#x20;

```
ExampleItem4:
  material: STONE
  lore:
  - '物品使用次数: %neigeitems_charge%/%neigeitems_maxCharge%'
  options:
    charge: 10
```

options下的所有配置项, 即为"额外选项"
