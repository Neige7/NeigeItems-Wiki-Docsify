# 安装

## 需求

1.12.2-1.19.3 bukkit服务端

已测试：
* paper1.12.2-1.19.3
* arclight1.16.5
* spigot1.12.2
* catserver1.12.2

## 安装NeigeItems

[NeigeItems-自动构建](https://github.com/Neige7/NeigeItems-Kotlin/actions)

1. 点击上方链接，通过GitHub下载NeigeItems插件
2. 将文件丢入plugins文件夹
3. 重启服务器

## 注意

!> 不要尝试通过Plugman等热重载工具加载NeigeItems
