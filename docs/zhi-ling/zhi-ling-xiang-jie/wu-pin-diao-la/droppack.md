# dropPack

* \[物品ID] NI物品包ID
* \[数量] 获取的数量，默认为1
* \[世界名] 物品掉落世界的名称
* \[X坐标] 物品掉落世界的X轴坐标
* \[Y坐标] 物品掉落世界的Y轴坐标
* \[Z坐标] 物品掉落世界的Z轴坐标
* (物品解析对象) 用于物品解析的玩家ID\
  &#x20;                        用于解析物品内的PAPI变量及随机节点
